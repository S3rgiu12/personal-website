## How to Run Locally

1. Clone the repository:
    ```bash
    git clone https://github.com/username/personal-website.git
    cd personal-website
    ```
2. Open the `index.html` file in your browser:
    ```bash
    open website.html
    ```

3. (If using a backend or need to serve it locally):
    ```bash
    npm install
    npm start
    ```
